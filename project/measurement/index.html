Distance measurement of objects in overlapping fields of view using binocular cameras based on MATLAB

Aug 27, 2020

This article develops a code implementation for ranging objects in overlapping fields of view using a binocular camera in the MATLAB environment.

The overall architecture of this article is as follows:

  1. Based on the provided calibration images, use the Stereo Camera Calibrator toolbox in MATLAB to calibrate the internal and external parameters of the left and right cameras.

  2. Using the parameters obtained in the first step, remove the distortion of the left and right ranging images.

  3. Perform Bouguet polar correction to convert non parallel binocular systems into aligned parallel binocular systems.

  4. For a parallel binocular system, select the same pixel on the left and right images and use the ranging formula to calculate the distance
